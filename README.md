# Лабораторная работа №1 — Вариант 10

Решения оформлены в одном классе `Lab1Var10` с методами для каждой задачи и интерактивной программой `Main1Var10`, которая вызывает все методы по очереди.

## Задание 1. Методы с числами и символами

### Задача 1.1 — Букву в число (`int charToNum(char x)`)

Суть: преобразовать символ '0'..'9' в число 0..9.
Алгоритм: вычесть `'0'` из символа.
**Тесты:**

* Ввод: '5' → Вывод: 5
* Ввод: '0' → Вывод: 0

### Задача 1.2 — Есть ли позитив (`boolean isPositive(int x)`)

Суть: проверить, положительное ли число.
Алгоритм: `x > 0`.
**Тесты:**

* Ввод: 3 → Вывод: true
* Ввод: -7 → Вывод: false

### Задача 1.3 — Большая буква (`boolean isUpperCase(char x)`)

Суть: проверить, является ли символ заглавной латинской буквой.
Алгоритм: `'A' <= x <= 'Z'`.
**Тесты:**

* Ввод: 'B' → true
* Ввод: 'g' → false

### Задача 1.4 — Делитель (`boolean isDivisor(int a, int b)`)

Суть: проверка, делится ли одно число на другое нацело.
Алгоритм: `(a % b == 0) || (b % a == 0)`, исключая ноль.
**Тесты:**

* Ввод: 6, 3 → true
* Ввод: 5, 2 → false

### Задача 1.5 — Сумма последних цифр (`int lastNumSum(int a, int b)`)

Суть: сумма последних цифр двух чисел.
Алгоритм: `(a % 10) + (b % 10)`.
**Тесты:**

* Ввод: 12, 35 → 7
* Ввод: 7, 9 → 16

## Задание 2. Математика

### Задача 2.1 — Модуль числа (`int abs(int x)`)

Суть: вернуть абсолютное значение числа.
Алгоритм: `x < 0 ? -x : x`.
**Тесты:**

* Ввод: -5 → 5
* Ввод: 7 → 7

### Задача 2.4 — Строка сравнения (`String makeDecision(int x, int y)`)

Суть: сравнить два числа и вернуть строку `"x<y"`, `"x>y"` или `"x==y"`.
**Тесты:**

* Ввод: 3,5 → "3<5"
* Ввод: 4,4 → "4==4"

### Задача 2.5 — Тройной максимум (`int max3(int x, int y, int z)`)

Суть: найти максимум из трёх чисел.
**Тесты:**

* Ввод: 1,9,3 → 9
* Ввод: -1, -5, -3 → -1

### Задача 2.7 — Двойная сумма (`int sum2(int x, int y)`)

Суть: сумма двух чисел, если сумма между 10 и 19 включительно → вернуть 20.
**Тесты:**

* Ввод: 7,5 → 20
* Ввод: 3,4 → 7

### Задача 2.9 — День недели (`String day(int x)`)

Суть: вернуть день недели по номеру (1..7).
**Тесты:**

* Ввод: 1 → "понедельник"
* Ввод: 7 → "воскресенье"

**Тесты:**
